# Generador de contraseñas seguras

Un proyecto sencillo para poder generar contraseñas seguras, permitiendo elegir
la cantidad de contraseñas, el tamaño y si se requiere que se agreguen números
y caracteres especiales.

## Stack Tecnológico

* Lenguajes: Python3.
* Microframework: Flask
    * flask_cors: Permite el acceso a recursos de la API Flask desde dominios
      diferentes.

# Entorno producción

[Producción](https://passwordgenerator.fjdev.icu/)

El proyecto esta desplegado en un servidor VPS utilizando:

* Web Server: Ngix.
* Aplication Server: Gunicorn
* SSL
